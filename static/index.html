La red

Una sola red para las apps de ayuda

Muchas apps reconstruyen el mismo listado de desaparecidos, centros de acopio y hospitales por separado. Aquí se conectan a un índice común con una sola API, en lugar de repetir el trabajo cada una por su cuenta.

Cómo te conectas

Tres formas de conectarte

01 — Te jalamos

Expones un endpoint JSON de lectura y lo registras con el mapeo de tus campos. Nosotros lo sincronizamos.

Registrar tu fuente →

02 — Tú empujas

Tu app envía los datos ya en el esquema común a la API de ingesta (POST /api/ingest), con tu clave, cuando quieras.

Ver cómo integrar →

03 — Federación

Nodos que se sincronizan entre sí vía /api/peers. Lo que indexa uno queda disponible para los demás.

Ver la red →
